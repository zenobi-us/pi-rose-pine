## Research Output

**Primary Research Document:** [Complete Rose Pine Theme Research](research-ff4d0949-rose-pine-theme-complete.md)

### Key Findings

1. **Official Sources:**
   - Website: https://rosepinetheme.com/
   - GitHub: https://github.com/rose-pine
   - NPM: @rose-pine/palette

2. **Three Variants:**
   - Rosé Pine (Main) - Primary dark variant
   - Rosé Pine Moon - Deeper dark variant  
   - Rosé Pine Dawn - Light variant

3. **15 Colours Per Variant:**
   - 9 neutral colours (base, surface, overlay, muted, subtle, text, highlight low/med/high)
   - 6 accent colours (love, gold, rose, pine, foam, iris)

4. **Design Philosophy:**
   - "All natural pine, faux fur and a bit of soho vibes for the classy minimalist"
   - Minimalist aesthetic with natural inspiration
   - 200+ ports across various applications

5. **Implementation Resources:**
   - Official template repository
   - Build tool (Go-based)
   - Palette tool (@rose-pine/palette npm package)
   - Brand assets and social image generator
