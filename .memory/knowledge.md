**Official Website:** https://rosepinetheme.com/  
**GitHub:** https://github.com/rose-pine  
**NPM Package:** @rose-pine/palette

### Theme Variants
1. **Rosé Pine (Main)** - Primary dark variant
2. **Rosé Pine Moon** - Deeper dark variant
3. **Rosé Pine Dawn** - Light variant

### Core Philosophy
"All natural pine, faux fur and a bit of soho vibes for the classy minimalist"

### Colour Structure
- **15 colours per variant**
- **9 neutral colours** (base, surface, overlay, muted, subtle, text, highlight low/med/high)
- **6 accent colours** (love, gold, rose, pine, foam, iris)

### Key Implementation Points
- Use exact hex values from official specifications
- Follow semantic colour roles
- Support all three variants
- Consider using @rose-pine/palette package
- Maintain minimalist design philosophy
- Follow official template structure for consistency

## Key Insights

### Design Principles
1. **Minimalism**: Clean, curated 15-colour palette
2. **Natural inspiration**: Pine, rose, foam elements
3. **Consistency**: 200+ ports with unified design
4. **Accessibility**: Three variants for different conditions

### Technical Details
- Colour format: Hex values (without # in v4+)
- Variants: main, moon, dawn
- File naming: snake-case (rose-pine, rose-pine-moon, rose-pine-dawn)
- Official palette tool provides multiple export formats (CSS, Tailwind, JS/TS)
