### Theme Variants
1. **Rosé Pine (Main)** - Primary dark variant (warm, cozy tones)
2. **Rosé Pine Moon** - Deeper dark variant (richer contrast)
3. **Rosé Pine Dawn** - Light variant (inverted for daylight)

### Core Philosophy
"All natural pine, faux fur and a bit of soho vibes for the classy minimalist"

### Colour Structure
- **15 colours per variant** (precisely curated)
- **9 neutral colours:** base, surface, overlay, muted, subtle, text, highlight_low, highlight_med, highlight_high
- **6 accent colours:** love (red), gold (yellow), rose (cyan), pine (green), foam (blue), iris (magenta)

### Key Implementation Points
- ✅ Use @rose-pine/palette package (not hardcoded values)
- ✅ Implement all three variants (required for community acceptance)
- ✅ Use exact hex values (pixel-perfect accuracy expected)
- ✅ Follow semantic colour roles (base/surface/overlay/text/etc.)
- ✅ Apply minimalist design philosophy
- ✅ Map ANSI terminal colors correctly
- ✅ Follow official template structure
- ✅ Consider gallery submission when complete
